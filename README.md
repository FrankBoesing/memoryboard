# memoryboard

1..6x spi jedec-memory soic-8 (e.g. 23LC1024 RAM)

Partslist:

- 6 x SPI jedec Memory of your choice (SOIC-8) (RAM (e.g. 23LC1024) , Flash...)
- 1 x 74LCX126MX SOIC-14
- 1 x 74LCX138MX SOIC-16
- 7 x Capacitor 0.1uF 0603
- 3 x Resistor 10K 0603
